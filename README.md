Overview
This repository contains the code for Race Car Dashboard Prototype v1, designed to prioritize minimalist clarity and at-a-glance performance data. This prototype was part of a larger project exploring how dashboard design impacts driver performance in racing simulations.

Key Features
Minimalist Design: Clean layout with essential information only.
At-a-Glance Data: Key metrics (speed, RPM, gear) are prominently displayed.
HTML, CSS, and JavaScript Implementation.
